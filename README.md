# ztkyn-java

#### 介绍
Java 工具代码库，分享常用的代码功能，以及一些开源项目的使用教程。

#### 使用方法
> 不提供jar，Maven仓库发布,支持根据自己的需求魔改项目，不受代码本来限制,不违反相关法律或者授权许可
1. clone 项目到本地，将代码打包到本地仓库或者私有仓库
2. 将代码集成到项目中
